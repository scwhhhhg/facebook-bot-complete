# Roadmap (2026)

Roadmap ini ditulis dari sudut pandang “apa yang paling menambah nilai untuk pembeli”.

## A) Dashboard (Ops Panel)
- Daftar akun + status bot (running/failed/idle)
- Queue content + approval workflow (draft → review → publish)
- Tombol pause/resume per akun atau per bot

## B) Analytics \u0026 Reporting
- Ringkasan aktivitas harian/mingguan per akun
- Export laporan (CSV/Google Sheets)
- Rekomendasi jam posting berbasis hasil

## C) Guardrails (Responsible Automation)
- Smart rate limiter adaptif per bot
- Deteksi repetisi konten/media lebih kuat (hash-based)
- Template safe-mode untuk akun baru

## D) Integrations
- Webhook (Zapier/Make)
- Airtable/Sheets sebagai content source
- Notifikasi ke Discord selain Telegram

Jika kamu pembeli dan ingin prioritas tertentu, request fitur lewat support channel (fitur yang paling banyak diminta akan diprioritaskan).
