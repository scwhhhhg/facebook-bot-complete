# FAQ

## Apakah ini aman?
Tidak ada automation yang “bebas risiko”. Karena itu, fokusnya adalah **responsible automation**: limit, random delay, quiet hours, dan monitoring real-time supaya kamu bisa stop cepat jika ada tanda-tanda masalah.

## Bisa jalan 24/7?
Bisa, paling ideal di VPS + process manager (mis. PM2). Kalau di PC rumah, pastikan koneksi stabil dan device tidak sleep.

## Perlu API key?
Untuk fitur AI (komentar/caption/gambar), iya. Untuk bot non-AI, bisa jalan tanpa AI key.

## Kenapa repo ini tidak bisa dipakai langsung?
Repo ini hanya untuk **preview/promosi**. Versi full (operasional) disediakan lewat landing page setelah pembelian lisensi.

## Apakah ada update?
Ada update berkala untuk perbaikan stabilitas, perubahan UI Facebook, dan penambahan fitur yang paling banyak diminta.

## Bisa request fitur?
Bisa. Request yang paling banyak diminta biasanya diprioritaskan di roadmap.
